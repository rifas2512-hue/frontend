# InternTrack - Internship Management Portal

## Overview

**InternTrack** is a simple web-based Internship Management Portal developed using HTML and CSS. It helps students track their internship activities, assigned tasks, progress, profile information, and certificate status through a clean and user-friendly interface.

---

## Features

### Dashboard

Displays internship statistics:

* Active Internships: **1**
* Tasks Completed: **8**
* Pending Tasks: **2**

### Student Profile

Shows student details including:

* Name: **Rifa Shaikh**
* Roll Number: **47**
* Branch: **Computer Technology**
* Company: **Rich System Solutions Pvt Ltd**
* Profile Image

### Assigned Tasks

Displays internship tasks in a table format with:

| Task              | Deadline | Status    |
| ----------------- | -------- | --------- |
| Website Design    | 15 June  | Completed |
| Report Submission | 19 June  | Pending   |
| Presentation      | 20 June  | Pending   |

### Internship Progress

Shows internship completion status:

* Progress: **30% Completed**

### Certificate Section

Provides internship completion information and a certificate download button.

4. Open `index.html` in any modern web browser.
